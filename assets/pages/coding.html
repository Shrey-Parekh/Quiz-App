Python

Python is a high-level, versatile programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

  • Easy to learn and read
  • Large standard library
  • Wide range of applications (web development, data science, AI)
  • Strong community support
  • Cross-platform compatibility
Python Example
# A simple Python program to calculate factorial
def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n-1)

# Calculate factorial of 5
result = factorial(5)
print(f"The factorial of 5 is {result}")

JavaScript

JavaScript is a versatile, high-level programming language that powers the interactive web. Originally created to make web pages dynamic, it has evolved into a powerful language used for both front-end and back-end development.

  • Essential for web development
  • Event-driven programming
  • Asynchronous capabilities
  • Rich ecosystem (Node.js, React, Vue)
  • Dynamic typing and prototypal inheritance
JavaScript Example
// A simple JavaScript program to create a counter
let count = 0;

function incrementCounter() {
    count++;
    return count;
}

// ES6 Arrow Function
const displayCounter = () => {
    console.log(`Current count: ${count}`);
};

// Increment the counter three times
incrementCounter();
incrementCounter();
incrementCounter();
displayCounter(); // Outputs: Current count: 3
